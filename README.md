# Neural Network from Scratch

## Overview

This project implements a feedforward neural network from scratch using Python and NumPy. Different activation functions and optimization strategies were implemented to better understand the fundamentals of deep learning.

## Features

- Feedforward neural network
- Backpropagation algorithm
- Sigmoid activation function
- ReLU activation function
- Adam optimizer
- California Housing dataset experiments

## Technologies

- Python
- NumPy
- Pandas

## Project Structure

```
Neural-Network/
│
├── housing.csv
├── main.py
├── mainsigmoid.py
├── mainrelu.py
└── mainadam.py
```

## Learning Outcomes

- Neural network implementation without deep learning frameworks
- Forward and backward propagation
- Gradient descent optimization
- Effect of different activation functions
